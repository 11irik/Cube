# Cube

Cube is a test class.

## Usage
```
input.txt file must contain 8 point which form right cube. 
The first four points form the first "tier" (a, b, c, d), 
and the second four form the second "tier" (a1, b1, c1, d1) in the same order. 
Points' coords must be written with a comma.
There is an argument exception, if the cube is incorrect.
Cubes can be written line by line.
```
